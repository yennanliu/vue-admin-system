# vue-admin-system

## Project setup

```bash
npm install
npm install express fs
```

### Compiles and hot-reloads for development

```bash

# FE app
npm run serve
# - Local:   http://localhost:8080/

# BE app (server)
node src/server.js
# Server is running at http://localhost:3000
```

### Compiles and minifies for production

```bash
npm src/run build
```

### Lints and fixes files

```bash
npm run lint
```
